Test tool for findPathBetweenSmarts function

GenX: O=C(O)C(F)(F)C(OC(F)(F)F)(C(F)(F)C(F)(F)F)C(C(F)(F)F)(C(F)(F)F)C(F)(F)C(F)(F)F
6:2 FTOH: OCCCC(F)(F)C(F)(C(F)(F)F)C(C(F)(F)F)(C(F)(F)F)C(F)(F)F
HFO: FC(=C(F)C(C(F)(F)F)(C(F)(F)C(F)(F)F)C(C(F)(F)F)(C(F)(F)F)C(F)(F)C(F)(F)F)C(F)(F)F

How It Works:

  • Perfluoroalkyl: C-C chains where all carbons are fully fluorinated: -CF2-CF2-
  • Polyfluoroalkyl: C-C chains with mix of F, H, Cl, Br, I: -CF2-CH2- or -CF2-CHF-
  • The function finds shortest paths between functional groups that go through fluorinated chains
  • Empty SMARTS 2 will search for paths to any fluorinated chain end
