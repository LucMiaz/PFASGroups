Test Export Formats Documentation

Summary CSV Export

The Summary CSV format provides:

  • One row per molecule
  • All matched PFAS groups in the same row (separated by semicolons)
  • Count of detected groups
  • Optional: Separate Yes/No columns for each unique PFAS group (when checkbox is checked)

Summary CSV Columns:

Input, Formula, [Original Columns], PFAS_Groups_Count, PFAS_Groups
    OR
Input, Formula, [Original Columns], PFAS_Groups_Count, Has_GroupA, Has_GroupB, ...

Example Summary CSV (semicolon-separated groups):

Input,Formula,PFAS_Groups_Count,PFAS_Groups
OCCCC(F)(C(F)(F)F)C(F)(F)F,C7H5F9O,2,FTOHs; semi-fluoroalkyl ketone
FC(F)C(F)(F)F,C3F6,0,

Example Summary CSV (separate columns):

Input,Formula,PFAS_Groups_Count,Has_FTOHs,Has_semi-fluoroalkyl ketone
OCCCC(F)(C(F)(F)F)C(F)(F)F,C7H5F9O,2,Yes,Yes
FC(F)C(F)(F)F,C3F6,0,No,No

Detailed CSV Export

The Detailed CSV format provides:

  • One row per match (if a molecule has multiple PFAS groups, it will have multiple rows)
  • One row per chain (if a match has multiple chains/pathways, it will have multiple rows)
  • Pathway type information (Perfluoroalkyl, Polyfluoroalkyl, etc.)
  • Chain length and atom indices
  • Perfluoroalkyl chain lengths (if applicable)

Detailed CSV Columns:

Input, Formula, [Original Columns], Group_ID, Group_Name, Group_Alias, Match_Count, 
Perfluoroalkyl_Chain_Lengths, Pathway_Type, Chain_Length, Chain_Atoms

Example Detailed CSV:

Input,Formula,Group_ID,Group_Name,Group_Alias,Match_Count,Perfluoroalkyl_Chain_Lengths,Pathway_Type,Chain_Length,Chain_Atoms
OCCCC(F)(C(F)(F)F)C(F)(F)F,C7H5F9O,15,fluorotelomer alcohols,FTOHs,1,4; 5,Perfluoroalkyl,4,12; 13; 14; 15
OCCCC(F)(C(F)(F)F)C(F)(F)F,C7H5F9O,15,fluorotelomer alcohols,FTOHs,1,4; 5,Polyfluoroalkyl,5,11; 12; 13; 14; 15
OCCCC(F)(C(F)(F)F)C(F)(F)F,C7H5F9O,28,Semi-fluoroalkyl ketones,semi-fluoroalkyl ketone,1,,,
FC(F)C(F)(F)F,C3F6,,,No PFAS groups detected,,,,,

Key Differences:

Feature Summary CSV Detailed CSV
Rows per molecule Always 1 1 or more (depends on matches and chains)
Group information Combined in one cell Separate columns
Pathway information Not included Included
Chain details Not included Included (length, atoms)
Best for Quick overview, filtering Detailed analysis, understanding pathways

Usage Tips:

  • Use Summary CSV when you need a compact overview and want to filter/sort molecules
  • Use Detailed CSV when you need to analyze pathway types or chain information
  • For the Summary CSV, check "Include all detected PFAS groups in separate columns" if you want binary Yes/No columns for each group (useful for data analysis)
  • The Detailed CSV is useful for understanding which specific fluorinated chains were detected and their properties
