Testing and Benchmarking

This section documents the testing methodology and benchmarking results for PFASgroups.

Test Suite

The PFASgroups package includes a comprehensive test suite covering:

  • Unit tests for individual functions

  • Integration tests for complete workflows

  • Validation against known PFAS databases

  • Performance benchmarks

Running Tests

# Run all tests
pytest tests/

# Run with coverage
pytest --cov=PFASgroups tests/

# Run specific test file
pytest tests/test_core.py

# Run with verbose output
pytest -v tests/

├── test_core.py           # Core parsing functions

Validation Methodology

OECD Test Set

The algorithm was validated against the OECD PFAS classification reference set:

import pandas as pd
from PFASgroups import parse_smiles

# Load OECD reference data
oecd_ref = pd.read_csv("oecd_reference.csv")

# Test each compound
results = []
for idx, row in oecd_ref.iterrows():
    parsed = parse_smiles([row['SMILES']])
    detected_groups = [g.name for g, _, _, _ in parsed[0]]

    expected = row['Expected_Group']
    match = expected in detected_groups
    results.append({'SMILES': row['SMILES'],
                   'Expected': expected,
                   'Detected': detected_groups,
                   'Match': match})

# Calculate accuracy
df = pd.DataFrame(results)
accuracy = df['Match'].mean()
print(f"OECD Test Accuracy: {accuracy:.2%}")

Unit Tests

Core Parsing Tests

import pytest
from PFASgroups import parse_smiles

def test_pfca_detection():
    """Test PFCA (perfluoroalkyl carboxylic acid) detection."""
    pfoa = "FC(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(=O)O"
    results = parse_smiles([pfoa])

    groups = [g.name for g, _, _, _ in results[0]]
    assert "PFCA" in groups

def test_pfsa_detection():
    """Test PFSA (perfluoroalkyl sulfonic acid) detection."""
    pfos = "FC(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)S(=O)(=O)O"
    results = parse_smiles([pfos])

    groups = [g.name for g, _, _, _ in results[0]]
    assert "PFSA" in groups

def test_chain_length():
    """Test chain length calculation."""
    # C4 PFCA
    pfba = "FC(F)(F)C(F)(F)C(F)(F)C(=O)O"
    results = parse_smiles([pfba])

    for g, count, chains, matched in results[0]:
        if g.name == "PFCA":
            assert chains == [3]  # 3 CF2 units

def test_multiple_groups():
    """Test detection of multiple groups."""
    # Compound with both acid and sulfonate
    smiles = "OC(=O)C(F)(F)C(F)(F)C(F)(F)C(F)(F)S(=O)(=O)O"
    results = parse_smiles([smiles])

    groups = [g.name for g, _, _, _ in results[0]]
    assert len(groups) >= 2

Fingerprint Tests

from PFASgroups import generate_fingerprint
import numpy as np

def test_fingerprint_shape():
    """Test fingerprint array shape."""
    smiles = ["FC(F)(F)C(=O)O", "FC(F)(F)S(=O)(=O)O"]
    fps, info = generate_fingerprint(smiles)

    assert fps.shape[0] == 2  # 2 molecules
    assert fps.shape[1] > 0   # At least 1 group

def test_binary_fingerprint():
    """Test binary fingerprint values."""
    smiles = ["FC(F)(F)C(=O)O"]
    fps, info = generate_fingerprint(smiles, count_mode='binary')

    assert np.all((fps == 0) | (fps == 1))

def test_count_fingerprint():
    """Test count fingerprint values."""
    smiles = ["FC(F)(F)C(=O)O"]
    fps, info = generate_fingerprint(smiles, count_mode='count')

    assert np.all(fps >= 0)

Constraint Tests

from PFASgroups import PFASGroup

def test_eq_constraint():
    """Test exact equality constraint."""
    group = PFASGroup(
        id=1, name="Test",
        smarts1="C(=O)O",
        constraints={"eq": {"O": 2}}
    )

    assert group.constraint_eq({"O": 2, "C": 4})
    assert not group.constraint_eq({"O": 3, "C": 4})

def test_gte_constraint():
    """Test greater-than-or-equal constraint."""
    group = PFASGroup(
        id=1, name="Test",
        smarts1="C(=O)O",
        constraints={"gte": {"F": 3}}
    )

    assert group.constraint_gte({"F": 5})
    assert group.constraint_gte({"F": 3})
    assert not group.constraint_gte({"F": 2})

def test_only_constraint():
    """Test element restriction constraint."""
    group = PFASGroup(
        id=1, name="Test",
        smarts1="C(=O)O",
        constraints={"only": ["C", "F", "O", "H"]}
    )

    assert group.constraint_only({"C": 4, "F": 7, "O": 2, "H": 1})
    assert not group.constraint_only({"C": 4, "F": 7, "O": 2, "S": 1})

import time
from PFASgroups import parse_smiles

def benchmark(smiles_list):
    """Benchmark parsing performance."""
    start = time.time()
    results = parse_smiles(smiles_list)
    elapsed = time.time() - start

    print(f"Total: {elapsed:.2f}s")
    print(f"Per molecule: {elapsed/len(smiles_list)*1000:.2f}ms")
    print(f"Throughput: {len(smiles_list)/elapsed:.0f} mol/s")

Continuous Integration

The package uses GitHub Actions for CI/CD:

# .github/workflows/tests.yml
name: Tests
on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: [3.8, 3.9, '3.10', '3.11']

    steps:
      - uses: actions/checkout@v3
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: ${{ matrix.python-version }}
      - name: Install dependencies
        run: |
          pip install -e .[dev]
      - name: Run tests
        run: pytest --cov=PFASgroups

Known Limitations

  1. Stereochemistry: Not considered in current version

  2. Tautomers: Some tautomeric forms may not be recognized

  3. Salts: Salt forms may need neutralization before analysis

  4. Polymers: Polymer structures not fully supported

  5. Radicals: Radical species not handled
