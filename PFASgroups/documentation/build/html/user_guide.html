User Guide

This comprehensive guide covers all aspects of using PFASgroups for PFAS analysis.

Basic Usage

Parsing SMILES Strings

The primary function for PFAS analysis is parse_smiles:

from PFASgroups import parse_smiles

# Single SMILES
results = parse_smiles("FC(F)(F)C(F)(F)C(=O)O")

# Multiple SMILES as a list
smiles_list = [
    "FC(F)(F)C(F)(F)C(=O)O",      # PFOA
    "FC(F)(F)C(F)(F)S(=O)(=O)O",  # PFHxS
    "C(C(F)(F)F)F"                 # HFC
]
results = parse_smiles(smiles_list)

Working with RDKit Molecules

You can also parse RDKit molecule objects directly:

from PFASgroups import parse_groups_in_mol
from rdkit import Chem
from rdkit.Chem.rdMolDescriptors import CalcMolFormula

mol = Chem.MolFromSmiles("FC(F)(F)C(F)(F)C(=O)O")
formula = CalcMolFormula(mol)

results = parse_groups_in_mol(mol, formula=formula)

Component-Based Analysis

For cyclic or complex structures, use component-based analysis:

# Standard chain-based analysis
results = parse_smiles(smiles_list, bycomponent=False)

# Component-based analysis (better for cyclic structures)
results = parse_smiles(smiles_list, bycomponent=True)

The bycomponent=True option finds connected components of fluorinated atoms rather than chains between functional groups. This is particularly useful for:

  • Cyclic PFAS compounds

  • Highly branched structures

  • Compounds with multiple fluorinated regions

Output Formats

PFASgroups supports multiple output formats:

# Default: list of tuples
results = parse_smiles(smiles_list, output_format='list')

# Pandas DataFrame
df = parse_smiles(smiles_list, output_format='dataframe')
print(df.columns)  # ['smiles', 'group_id', 'group_name', 'match_count', ...]

# CSV string
csv_str = parse_smiles(smiles_list, output_format='csv')

Fingerprint Generation

Binary Fingerprints

Generate binary fingerprints for presence/absence of PFAS groups:

from PFASgroups import generate_fingerprint

smiles_list = ["FC(F)(F)C(F)(F)C(=O)O", "C(C(F)(F)F)F"]

# Binary vector (default)
fps, info = generate_fingerprint(smiles_list)
print(fps.shape)  # (2, 55)
print(info)       # Group information

Count-Based Fingerprints

Count the number of matches for each group:

fps, info = generate_fingerprint(smiles_list, count_mode='count')

Chain Length Fingerprints

Use maximum chain length as the fingerprint value:

fps, info = generate_fingerprint(smiles_list, count_mode='max_chain')

Selecting Specific Groups

Generate fingerprints for specific groups only:

# By ID range
fps, info = generate_fingerprint(smiles_list, selected_groups=range(1, 29))

# By specific IDs
fps, info = generate_fingerprint(smiles_list, selected_groups=[1, 6, 14, 22])

# Generic groups only
fps, info = generate_fingerprint(smiles_list, selected_groups=range(29, 58))

Representation Formats

# NumPy vector (default)
fps, info = generate_fingerprint(smiles_list, representation='vector')

# Dictionary format
fps, info = generate_fingerprint(smiles_list, representation='dict')
# Returns: [{'PFCAs': 1, 'carboxylic acid': 1}, ...]

# Sparse format (only non-zero)
fps, info = generate_fingerprint(smiles_list, representation='sparse')

Machine Learning Integration

Example with scikit-learn:

from PFASgroups import generate_fingerprint
from sklearn.ensemble import RandomForestClassifier
import numpy as np

# Generate fingerprints
smiles = ["FC(F)(F)C(F)(F)C(=O)O", "C(C(F)(F)F)F", ...]
labels = [1, 0, ...]  # Your labels

fps, info = generate_fingerprint(smiles)

# Train classifier
clf = RandomForestClassifier()
clf.fit(fps, labels)

Homologue Series Generation

Generate shorter-chain analogues by iteratively removing CF₂ moieties:

from PFASgroups import generate_homologues
from rdkit import Chem

# Start with PFOA (C8)
mol = Chem.MolFromSmiles("FC(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(F)(F)C(=O)O")

# Generate homologues
homologues = generate_homologues(mol, smartsPathName='Perfluoroalkyl')

# Returns shorter chain analogues:
# PFOA (C8) → PFHpA (C7) → PFHxA (C6) → PFPeA (C5) → PFBA (C4)
for inchikey, formulas in homologues.items():
    for formula, h_mol in formulas.items():
        print(f"{formula}: {Chem.MolToSmiles(h_mol)}")

For polyfluoroalkyl compounds:

homologues = generate_homologues(
    mol,
    smartsPathName='Polyfluoroalkyl',
    repeating='C([F,H,I,Br,Cl])[F,H,I,Br,Cl]'
)

Visualization

Basic Plotting

from PFASgroups import plot_pfasgroups

# Simple plot
plot_pfasgroups("FC(F)(F)C(F)(F)C(=O)O")

# Multiple molecules
smiles_list = ["FC(F)(F)C(F)(F)C(=O)O", "FC(F)(F)C(F)(F)S(=O)(=O)O"]
plot_pfasgroups(smiles_list)

Customizing Output

# SVG output
plot_pfasgroups(smiles, svg=True, path="output.svg")

# Customize dimensions
plot_pfasgroups(smiles, subwidth=400, subheight=400)

# Grid layout
plot_pfasgroups(smiles_list, ncols=3)

# Show atom indices
plot_pfasgroups(smiles, addAtomIndices=True)

# Show bond indices
plot_pfasgroups(smiles, addBondIndices=True)

Batch Processing

Processing Large Datasets

For large datasets, use batch processing with progress bars:

from PFASgroups import parse_smiles
import pandas as pd
from tqdm import tqdm

# Load data
df = pd.read_csv("molecules.csv")
smiles_list = df['smiles'].tolist()

# Process in batches
batch_size = 1000
all_results = []

for i in tqdm(range(0, len(smiles_list), batch_size)):
    batch = smiles_list[i:i+batch_size]
    results = parse_smiles(batch)
    all_results.extend(results)

Using DataFrame Output

# Direct DataFrame output
df_results = parse_smiles(smiles_list, output_format='dataframe')

# Save to CSV
df_results.to_csv("pfas_analysis_results.csv", index=False)

# Filter by group
pfcas = df_results[df_results['group_name'] == 'Perfluoroalkyl carboxylic acids']

Working with PFAS Definitions

PFAS definitions provide broader classification criteria:

from PFASgroups import get_PFASDefinitions, PFASDefinition
from rdkit import Chem

# Get default definitions
definitions = get_PFASDefinitions()

# Check if a molecule matches a definition
mol = Chem.MolFromSmiles("FC(F)(F)C(F)(F)C(=O)O")

for defn in definitions:
    if defn.applies_to_molecule(mol):
        print(f"Matches: {defn.name}")
        print(f"  Description: {defn.description}")

Error Handling

PFASgroups handles common errors gracefully:

from PFASgroups import parse_smiles

# Invalid SMILES are handled
results = parse_smiles(["valid_smiles", "invalid##smiles", "another_valid"])
# Invalid SMILES return empty results

# Molecules with valence errors are fragmented
# The algorithm attempts to correct valence issues automatically

Advanced Topics

Custom Shortest Path Method

results = parse_smiles(
    smiles_list,
    shortest_path_method='dijkstra'  # or other NetworkX methods
)

Including PFAS Definitions

from PFASgroups import parse_mols

results = parse_mols(
    mol_list,
    include_PFAS_definitions=True  # Include broader PFAS definitions
)

Performance Tips

  1. Pre-compile SMARTS patterns: When using custom groups, patterns are cached after first use

  2. Batch processing: Process molecules in batches rather than one at a time

  3. Use RDKit molecules: If you already have RDKit Mol objects, use parse_groups_in_mol directly

  4. Select specific groups: Use selected_groups to limit analysis to relevant groups
